Property Value Description
Z-index z-index:1 The CSS z-index property specifies how far back or how far forward an element will appear on a web page when it overlaps other elements.
Static position position:static Default position in html.
Fixed position position:fixed When the CSS position has a value of fixed, it is set/pinned to a specific spot on a page. The fixed element stays the same regardless of scrolling. The navigation bar is a great example of an element that is often set to position:fixed;, enabling the user to scroll through the web page and still access the navigation bar.
Absolute position position:absolute Position the element relative to the closest parent element that has a relative or absolute positioning. Absolute will remove an element from the document flow. Use 'top', 'left', 'bottom' and 'right' to position the element anywhere in the document.
Relative position position:relative Place an element relatively to its default (original) position. You have to use offset properties like 'top' etc. to determine where it should position. Otherwise nothing will change (and the element acts as it has 'static' position).
Inline display display:inline Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted.
Block display display:block Block-level elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted.
Inline-block display display:inline-block Inline-block elements can appear next to each other, and can have their width and height manually adjusted.
Float float:right or float:left The CSS float property determines how far left or how far right an element should float within its parent element. The value left floats an element to the left side of its container and the value right floats an element to the right side of its container. For the property float, the width of the container must be specified or the element will assume the full width of its containing element.
Clear clear:left, clear:right, clear:both or clear:none The CSS clear property specifies how an element should behave when it bumps into another element within the same containing element.The clear is usually used in combination with elements having the CSS float property. This determines on which sides floating elements are allowed to float.
